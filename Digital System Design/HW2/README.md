## 題目 2：實作 8 bits 上下數器（從 20₁₆ 到 A0₁₆）

### 說明

正緣觸發

- 輸入 Input：CLK、RESET
- 模式 Mode：UP_DOWN
- 輸出 Output：Q

`UP_DOWN = 0` 時，上數。  
`UP_DOWN = 1` 時，下數。
